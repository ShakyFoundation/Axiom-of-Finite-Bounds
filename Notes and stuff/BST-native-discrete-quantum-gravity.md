## 2. BST-Native Causal Sets
### 2.1 Definition of Bounded Causal Set
**Definition:** A **bounded causal set** $\mathcal{C}$ is a finite partially ordered set $(C, \prec)$ where:
1.  **Domain:** $C$ is a finite set of elements (spacetime events), $|C| \leq n_M$.
2.  **Order:** $\prec$ is a strict partial order on $C$ (irreflexive, transitive).
3.  **Local Finiteness:** For any $x, y \in C$, the causal interval $I(x, y) = \{z \in C \mid x \prec z \prec y\}$ is finite.
    *   *BST Note:* Since $C$ is globally finite (AFB), local finiteness is automatic.
4.  **Causal Structure:** $x \prec y$ means "$x$ is in the causal past of $y$".

**BST Implementation:**
*   $C$ exists as a set by **Part VII, Definition 7.9** (Partial Orders).
*   $\prec$ exists as a relation subset of $C \times C$ (**Part VII, Definition 7.3**).
*   **Status:** Type I Recovery (exact BST definition).

$$ S_{BD}(\mathcal{C}) = \frac{1}{\ell_P^2} \sum_{x \in C} \left( c_0 N_0(x) - c_1 N_1(x) + c_2 N_2(x) - c_3 N_3(x) \right) $$
where $N_k(x)$ is the number of $k$-element chains ending at $x$.

**BST Implementation:**

*   Since $C$ is finite, all sums are finite.

*   **Status:** Type I Recovery.

### 4.2 Combinatorial Ricci Curvature

$$ \text{Ric}(x) \approx \frac{\delta S_{BD}}{\delta \text{Order}(x)} $$
